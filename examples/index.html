BS4Modal

Interactive examples for the BS4Modal jQuery plugin.

1. Declarative — data-attribute trigger

Add a title and data-content attribute to any element. Call .bs4Modal() on it once.

Open declarative modal 
<a href="#"
   title="Hello from data attributes"
   data-content="<p>Content from data-content attribute.</p>"
   class="js-declarative">Open</a>

$('.js-declarative').bs4Modal();
2. Static content — imperative

Pass content directly as a string. AJAX is skipped automatically.

$.bs4Modal({
    title: 'Static Content',
    content: '<p>This HTML was passed directly as a string.</p>'
});
3. No header, custom footer

Hide the header and provide custom footer HTML with action buttons.

$.bs4Modal({
    header: false,
    content: '<p class="mb-0">Are you sure you want to delete this item?</p>',
    footer: '<button class="btn btn-danger" id="confirm-delete">Delete</button>' +
            '<button class="btn btn-secondary" data-dismiss="modal">Cancel</button>'
});
4. Large modal with subtitle and title icon

Use modalsize, subtitle, and titleIcon options.

$.bs4Modal({
    title: 'User Profile',
    subtitle: 'View and edit account details',
    titleIcon: '<svg ...></svg>',
    modalsize: 'modal-lg',
    content: '...'
});
5. Manual close — programmatic dismiss

Set closemodal: 'manual' to remove all dismiss buttons. Close programmatically after a timed action.

var modal = $.bs4Modal({
    title: 'Processing...',
    content: '<p>Saving your changes. Closing automatically in 3 seconds.</p>',
    closemodal: 'manual',
    backdrop: 'static',
    keyboard: false
});

setTimeout(function() { modal.dismiss(); }, 3000);
6. Disable and re-enable closing

Prevent the user from closing during a simulated async operation, then restore normal behaviour.

var modal = $.bs4Modal({ title: 'Submitting Form', content: '...' });

modal.disableClosing(); // lock

setTimeout(function() {
    modal.setContent('<p>Done! You can now close this modal.</p>');
    modal.resetClosing(); // unlock
}, 3000);
7. Lifecycle callbacks

Hook into onOpenBefore, onOpen, onCloseBefore, and onClose. Check the browser console for output.

$.bs4Modal({
    title: 'Lifecycle Demo',
    content: '<p>Open the browser console to see callback output.</p>',
    onOpenBefore:  function() { console.log('[bs4Modal] onOpenBefore'); },
    onOpen:        function() { console.log('[bs4Modal] onOpen'); },
    onCloseBefore: function() { console.log('[bs4Modal] onCloseBefore'); },
    onClose:       function() { console.log('[bs4Modal] onClose'); }
});
8. Update content after open

Use setContent() and prependContent() on a live modal instance.

var modal = $.bs4Modal({ title: 'Live Update', content: '<p>Original content.</p>' });

// Replace content after 2s
setTimeout(function() {
    modal.setContent('<p>Content was replaced via setContent().</p>');
}, 2000);

// Prepend a notice after 4s
setTimeout(function() {
    modal.prependContent('<div class="alert alert-success">Prepended via prependContent()</div>');
}, 4000);
9. Auto close button in footer

Set closebtn: true to add a Cancel button to the footer without writing footer HTML manually.

$.bs4Modal({
    title: 'With Cancel Button',
    content: '<p>A Cancel button is added to the footer automatically.</p>',
    closebtn: true
});
10. Mobile full-screen breakdown

Below the mobileBreakdown breakpoint the modal fills the screen. Resize the browser window narrow to see the effect.

$.bs4Modal({
    title: 'Mobile Responsive',
    content: '<p>Resize the browser below 768px to see full-screen layout.</p>',
    mobileBreakdown: 768   // default is 576
});
11. Long content — default vs scrollable dialog

By default the entire modal grows with the content and the page scrolls. Set forceScrollableDialog: true to keep the modal at a fixed height and scroll inside the modal body instead. 

// Default — modal grows, page scrolls
$.bs4Modal({
    title: 'Long Content',
    content: '... lots of content ...',
    modalsize: 'modal-lg'
});

// Scrollable dialog — modal stays fixed height, body scrolls inside
$.bs4Modal({
    title: 'Long Content — Scrollable Dialog',
    content: '... lots of content ...',
    modalsize: 'modal-lg',
    forceScrollableDialog: true
});
Note: Examples 12 and 13 use AJAX and require the page to be served over HTTP. Opening this file directly as file:// will cause the requests to fail due to browser security restrictions. Serve the project locally and open it at a URL such as http://localhost:8080/examples/index.html.
12. AJAX — load content from a static file

The modal fetches partials/user-profile.html via AJAX and displays it in the body. A loading spinner is shown while the request is in flight.

Load user profile 
<a href="partials/user-profile.html" title="User Profile" class="js-ajax-profile">
    Load user profile
</a>

// Declarative — href drives the AJAX request automatically
$('.js-ajax-profile').bs4Modal({
    modalsize: 'modal-lg',
    cache: true
});
13. AJAX — load a form, simulate submit with closing lock

Fetches partials/edit-form.html via AJAX. Clicking Save in the footer disables closing, simulates a 2 second server round-trip, then shows a success message and re-enables closing. 

var modal = $.bs4Modal({
    href: 'partials/edit-form.html',
    title: 'Edit User',
    modalsize: 'modal-lg',
    footer: '<button id="btn-save-form" class="btn btn-primary">Save changes</button>' +
            '<button class="btn btn-secondary" data-dismiss="modal">Cancel</button>'
});

$(document).on('click', '#btn-save-form', function() {
    modal.disableClosing();
    modal.showLoading();
    // simulate server request
    setTimeout(function() {
        modal.setContent(
            '<div class="alert alert-success mb-0">Changes saved successfully.</div>'
        );
        modal.resetClosing();
    }, 2000);
});
